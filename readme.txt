Plugin serwisu Miniblog.pl. Zapewnia narzedzia do zarzadzania Twoja strona oraz jej poprawne dzialanie. Wylaczenie pluginu jest rownoznaczne z utrata wsparcia i jest naruszeniem regulaminu Miniblog.pl.

== Installation ==

Don't use it if your WordPress is not hosted on Miniblog.pl
